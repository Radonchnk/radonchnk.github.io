In this report I will explain the lifecycle of a Linux process within a general-purpose op-
erating system, using the command cp -vi .bashrc bashrc.bak as a practical example.
We will explore the fundamental concepts of the Linux operating system, including the
distinction between user space and kernel space, the role of the command line, and the
intricacies of process creation, memory management, file operations, and user interac-
tion. This report will delve into the security controls inherent in the process lifecycle,
examining file permissions, and the security boundaries enforced between user and kernel
space.

This is a coursework that was submitted as part of a Operating Systems for Cyber Security from Year 1 as a part of the Cyber Security Degree at University of Warwick
