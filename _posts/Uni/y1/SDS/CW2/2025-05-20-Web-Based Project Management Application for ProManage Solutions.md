> **Project Overview:** This report presents the development of a secure web-based project management application for **ProManage Solutions**, designed to enhance team collaboration and productivity.
{: .prompt-info }

Built using the Flask Python framework, the application incorporates secure authentication, role-based access control (RBAC), password security, and comprehensive activity logging. This document details the application’s architecture, implemented features, design and security decisions, additional functionalities, and development process.

This is a coursework that was submitted as part of a Secure Software Development module from Year 1 as a part of the Cyber Security Degree at University of Warwick
